## Project setup
Install dependencies
```
yarn install
```
Download games' backgrounds (only need to do it once)
```
node ./scripts/downloadBackgrounds.js
```
Then start the development server
```
yarn serve
```
## Compiles and minifies for production
```
yarn build
```

## Lints and fixes files
```
yarn lint
```
